BidBuzz — Online Auction Platform

BidBuzz is a modern web-based auction platform where users can buy and sell a wide range of items — from everyday goods to rare collectibles. Unlike a traditional secondhand-only auction site, BidBuzz is open to all categories, enabling sellers to reach a broader audience and buyers to discover diverse products.

🌟 Project Overview
BidBuzz is designed to provide a seamless online auction experience with a clean interface and essential bidding features. Users can:

List items for auction with clear descriptions and images.

Place bids in real time and track auction progress.

View complete bid history for transparency.

Manage listings, bidding, and user details through an intuitive UI.

The platform ensures a competitive yet user-friendly bidding process, making it suitable for individuals, small businesses, and collectors alike.

🚀 Key Features
Universal Auction Categories — Not limited to secondhand goods; supports all types of products.

Bid History — Full visibility of bidding activity for each item.

Detailed Item Descriptions — Rich information to help buyers make informed decisions.

User Accounts — Secure login and profile management.

Responsive Design — Works smoothly on desktops, tablets, and smartphones.

🛠 Technology Stack
Backend: ASP.NET Core MVC

Frontend: HTML, CSS, Bootstrap, JavaScript

Database: Microsoft SQL Server
